Atheme IRC Services

Lorum ipsum blah blah blah

ChanServ

ChanServ gives normal users the ability to maintain control of a channel, without the need of a bot. Channel takeovers are virtually impossible when a channel is registered with ChanServ. Registration is a quick and painless process. Once registered, the founder can maintain complete and total control over the channel.

Learn more about ChanServ

NickServ

NickServ allows users to register a nickname for use with ChanServ and stop others from using that nick. NickServ allows the owner of a nickname to disconnect a user from the network that is using their nickname.

Learn more about NickServ

MemoServ

MemoServ allows registered users to send memos to other registered users. Memos are a message stored by MemoServ which the recipient can then read whenever they come online.

Learn more about MemoServ

GroupServ

GroupServ allows for the creation of groups of users, who can share control over channels, read memos sent to the group, et cetera.

Learn more about GroupServ

OperServ

OperServ provides essential network management services, such as rounting manipulation and access management.

Learn more about OperServ

GameServ

GameServ provides games and tools for playing games on the network.

Learn more about GameServ

InfoServ

InfoServ provides a system for sending informational notices to the network's users. InfoServ may be used to send notices to users when they connect or while they are connected.

Learn more about InfoServ

HostServ

HostServ provides users a way to request a desired virtual hostname. HostServ also provides a way for operators to add virtual hostnames which users may have automatically activated at their leisure.

Learn more about HostServ

Atheme on Windows

Windows builds are experimental and unsupported. Use at your own risk!

These files must be extracted to C:\Atheme. If you want to install Atheme elsewhere, build Atheme from source.
