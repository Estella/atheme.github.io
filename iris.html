Iris

is an AJAX IRC client. It is a dropin replacement for qwebirc, and is designed to integrate with the Atheme IRC platform.

Python + AJAX

Iris is build from the ground up using Python and a frontend powered by the MooTools JavaScript library. This AJAX-powered web client means no need for Flash, Java or any browser plugins. IRC from the browser has never been easier.

Download

You can download Iris by either grabbing a copy of the git tarball or by installing git and cloning git://github.com/atheme/iris.git. For more information, see Iris' page on GitHub.

Atheme Integration

Iris is designed to integrate brilliantly with the Atheme IRC platform. It supports automatic channel listning through ALIS, and identification to services through the SASL protocol.
