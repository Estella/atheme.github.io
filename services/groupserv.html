GroupServ

GroupServ provides tools for managing groups of users and channels.

REGISTER

Registers a group with GroupServ.

Syntax: REGISTER !atheme-developers

DROP

Drops a group registration.

Syntax: DROP !atheme-developers

INFO

Displays information about a registered group.

Syntax: INFO !atheme-developers

JOIN

Joins an open group.

Syntax: JOIN !atheme-developers

HELP

Displays contextual help information.

Syntax: HELP

FLAGS

Sets flags on a user in a group.

Syntax: FLAGS !atheme-developers nenolod +*

SET

Sets various options on a group.

Syntax: SET !atheme-developers CHANNEL #atheme
