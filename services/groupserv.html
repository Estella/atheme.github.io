GroupServ

GroupServ provides tools for managing groups of users and channels.

REGISTER

Registers a group with GroupServ.

Syntax: 

REGISTER <group>

Example: 

REGISTER !atheme-developers

DROP

Drops a group registration.

Syntax: 

DROP <group>

Example: 

DROP !atheme-developers

INFO

Displays information about a registered group.

Syntax: 

INFO <group>

Example: 

INFO !atheme-developers

JOIN

Joins an open group.

Syntax: 

JOIN <group>

Example: 

JOIN !atheme-developers

HELP

Displays contextual help information.

Syntax: 

HELP [command]

Example: 

HELP FLAGS

FLAGS

Sets flags on a user in a group.

Syntax: 

FLAGS <group> <account> <flags>

Example: 

FLAGS !atheme-developers nenolod +*

SET

Sets various options on a group.

Syntax: 

SET <group> <key> [value]

Example: 

SET !atheme-developers CHANNEL #atheme

For more information (or help) with GroupServ, /msg GroupServ HELP on an Atheme instance.
