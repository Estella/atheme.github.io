Atheme-Web

Atheme meets the World Wide Web. Manage your channels, your user settings, even your network access list from Atheme-Web.

Python + WWW

Atheme-Web uses the Paste framework to provide a light-weight but well-polished backend and jQuery for a frontend guarenteed to delight the user.

Download

You can either download one of the preview release tarballs of Atheme-Web, or clone it using git from git://github.com/atheme/atheme-web.git.

Third-Party Interfaces

There are various alternative atheme-web type web applications. We know of AWP, an atheme web panel written in NodeJS. More will be noted as they start existing.

Please note: these are third-party applications. Atheme does not provide support for these applications.
